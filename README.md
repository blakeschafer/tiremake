# Terminal Industries — YOS Marketing Site

A remake of the [Terminal Industries](https://www.terminalindustries.com/) marketing site, rebuilt from scratch with modern tooling.

## Stack

- **Next.js 15** (App Router)
- **Tailwind CSS v4**
- **Framer Motion** (scroll-driven animations)
- **TypeScript**

## Features

- Scroll-driven hero animation: truck drives in, transitions from solid silhouette to wireframe with scan lines and detection brackets
- Character-by-character scroll text reveal
- Scroll-locked feature showcase
- Responsive layout with mobile nav

## Getting Started

```bash
npm install
npm run dev
```
